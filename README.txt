*create.sql:
	Consists of Drop Table Statements, Create Table statements and Insert Statements of all the five tables of our database.

*load.sql:
	Consists of eight queries, commands for indexing the tables and three Problematic queries.

*website.py:
	Consists of python code for website to our project. It is built using Streamlit library package ("https://streamlit.io/").

*user_info.csv:
	Consists of all the dummy data created for user_info table using online data generator tool ("https://www.onlinedatagenerator.com/").

*donor.csv:
	Consists of all the dummy data created for donor table using online data generator tool ("https://www.onlinedatagenerator.com/").

*recipient.csv:
	Consists of all the dummy data created for recipient table using online data generator tool ("https://www.onlinedatagenerator.com/").

*organ.csv:
	Consists of all the dummy data created for organ table using online data generator tool ("https://www.onlinedatagenerator.com/").

*health_history.csv:
	Consists of all the dummy data created for health_history table using online data generator tool ("https://www.onlinedatagenerator.com/").
